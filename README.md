# 高效慢活學官方網站

專業職涯諮詢服務品牌網站，基於 Next.js 15 構建。

## 🎯 專案特色

- ✅ **Next.js 15** - 最新版本 App Router
- ✅ **單頁式設計** - 流暢的使用者體驗
- ✅ **響應式佈局** - 完美支援手機與桌面
- ✅ **專業 SEO** - 完整的 Metadata 與結構化資料
- ✅ **互動組件** - 客戶見證與服務滑動展示

## 🎨 視覺風格

- **色調**：淡黃色 (#FFFBE0) + 大地色 (#e7b08e)
- **字體**：Inter 無襯線體
- **設計**：簡約專業，大量留白

## 🚀 快速開始

```bash
# 安裝依賴（需先安裝 Node.js）
npm install

# 啟動開發伺服器
npm run dev

# 在瀏覽器開啟
http://localhost:3000
```

## 🚀 部署到 Cloudflare Pages

### 方法一：連接 GitHub 倉庫（推薦）

1. 登入 Cloudflare Dashboard
2. 進入 `Pages` → `Create a project`
3. 連接 GitHub 帳號並選擇此倉庫
4. 設定建置配置：
   - **Framework preset**: Next.js
   - **Build command**: `npm run build`
   - **Build output directory**: `.next`
5. 點擊 `Save and Deploy`

### 方法二：直接上傳

詳細步驟請參閱 `DEPLOY_GUIDE.md`
